Binary Parameters

For the following options, please send a 1 to select. If you do not wish to receive this information, either send a 0 or omit the paramter from your request.

  • demoLoop
  • commercialFlights

Examples

  • http://72.78.75.212:9900/data?demoLoop=1
  • http://72.78.75.212:9900/data?demoLoop=1&commercialFlights=1&lngMin=-76.623080&lngMax=-73.828576&latMin=38.938079&latMax=40.632118
